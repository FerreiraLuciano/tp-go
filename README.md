## Features

- **Add** a new contact
- **List** all registered contacts
- **Update** a registered contact infos
- **Delete** a contact
- Add a contact **using flags**

## Install

```bash
  git clone https://github.com/ton-utilisateur/cli-crm-go.git
  cd cli-crm-go
```

---

## How to use

### 1. Launch the CLI

```bash
  go run main.go
```

### 2. Add a contact using flags

```bash
  go run main.go -name "Jean Dupont" -email "jean.dupont@example.com"
```
